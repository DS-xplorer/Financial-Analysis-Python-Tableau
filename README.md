# 📊 Financial Analysis Dashboard | Python

## 📌 Project Overview

This project demonstrates an end-to-end financial data analysis workflow using **Python** for data cleaning, preprocessing, exploratory data analysis (EDA), and KPI calculation, followed by **Tableau** for building an interactive financial dashboard.

The objective was to transform raw financial data into meaningful business insights that support data-driven decision-making.

---

## 🎯 Objectives

* Clean and preprocess messy financial data.
* Perform exploratory data analysis (EDA).
* Calculate key business KPIs.
* Build an interactive Tableau dashboard.
* Generate business insights and recommendations.

---
